🚚 Supply Chain Logistics & Inventory Efficiency Dashboard (Tableau Project)

📊 Overview

This project presents an interactive Tableau Dashboard designed to analyze and optimize supply chain, logistics, and inventory operations. It provides actionable insights into warehouse performance, inventory movement, transportation efficiency, and delivery reliability.

The primary goal is to enhance decision-making by identifying operational bottlenecks and improving overall supply chain efficiency.


---

🎯 Objectives

Monitor order fulfillment performance

Track inventory turnover and stock movement

Analyze transportation cost trends

Identify stockout risks

Measure on-time delivery performance

Compare warehouse efficiency



---

📌 Key KPIs

Order Fill Rate (%)

Inventory Turnover

Average Lead Time

Transportation Cost Per Unit

Warehouse Utilization (%)

On-Time Delivery (%)

Stockout Frequency (%)
